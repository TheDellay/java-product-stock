# 📦 Java Product Stock

Aplicação Java desenvolvida para gerenciamento de estoque de produtos utilizando conceitos de Programação Orientada a Objetos (POO).

## 📋 Funcionalidades

* Cadastro de produto
* Consulta de dados do produto
* Entrada de produtos no estoque
* Saída de produtos do estoque
* Cálculo automático do valor total em estoque
* Exibição formatada das informações utilizando o método `toString()`

## 🚀 Tecnologias Utilizadas

* Java
* Programação Orientada a Objetos (POO)
* IntelliJ IDEA

## 📂 Estrutura do Projeto

```text
src/
├── application/
│   └── Program.java
└── entities/
    └── Product.java
```

## 💻 Exemplo de Execução

```text
Enter product data:

Name: TV
Price: 900.00
Quantity in stock: 10

Product data:
TV, $ 900.00, 10 units, Total: $ 9000.00

Enter the number of products to be added in stock: 5

Updated data:
TV, $ 900.00, 15 units, Total: $ 13500.00

Enter the number of products to be removed from stock: 3

Updated data:
TV, $ 900.00, 12 units, Total: $ 10800.00
```
